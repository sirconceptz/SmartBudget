## Installation

### Prerequisites

- Flutter SDK (>= 3.x.x)
- Dart (>= 3.x.x)
- Android Studio or Xcode (for running the app)

### Steps

1. Clone this repository:
   ```bash
   git clone https://github.com/sirconceptz/BudgetManager.git
   cd BudgetManager
   ```

2. Install dependencies:
   ```bash
   flutter pub get
   ```

3. Run the application:
   ```bash
   flutter run
   ```

4. To build for iOS or Android, use:
   ```bash
   flutter build ios
   flutter build apk
   ```
